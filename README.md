# Разработка

Django application structure folder:
- annotation - основное приложение для анотирования.
- server - настройки проекта
- templates - базовые шаблоны проекта

Набор используемых инструментов и библиотек:
- Django
- Bootstrap 4.3.1
- Bootstrap-vue 2.0.1
- Vue 2.6.10
- Vue Router 3.1.3
- axios 0.19.0
- Font Awesome
- Bulma (кусок)
